# UI/UX Pro Max Library

This directory contains reusable components and design guidelines based on the "Liquid Glassmorphism" aesthetic.

## Contents
- **Components**: Pre-built interactive components with Framer Motion (Floating Navigation, Magnetic Cursor, Scroll hijacking).
- **Styles**: Global CSS setup with Tailwind v4, custom variables, and mesh gradient animations.
- **Assets**: Required fonts, placeholder images, and svg filters.

## How to use
When starting a new project, you can copy the `src/components` and `src/app/globals.css` into your project and use the `.agents/skills/ui-ux-pro-max` skill to instruct the AI to follow this exact design system.

This library is designed for Next.js, Tailwind CSS, and Framer Motion.
